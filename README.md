# Interpretable analysis of the factors influencing gold price movements in Egypt (2010-2026) using Power BI and predictive modeling

## Objective
Explain WHY gold prices move in Egypt by integrating multiple factors including economics, energy, crypto, geopolitical events, and seasonal trends.

## Data Sources
- Gold prices (2010–2026)
- USD/EGP, Forex rates
- Bitcoin and other cryptocurrencies
- Oil & Gas prices
- Global and local events
- Egyptian seasonal events (Ramadan, Eids, Weddings)

## Key Features
- Trend analysis of gold prices
- Correlation with economic and energy indicators
- Event-driven analysis (wars, holidays, global crises)
- Predictive modeling of gold and oil prices

## Dashboard Sections
1. Overview
2. Economic Impact
3. Crypto vs Gold
4. Egyptian Seasonal Events
5. Global Events
6. Forecasting

## Technologies Used
- Power BI / Dax / Power quire
- Excel / CSV / Sql 
- GitHub for version control
